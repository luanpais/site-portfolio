MINHAS COMPETÊNCIAS.

Lógica de programação

Neste curso introdutório de programação, mergulhei na lógica de programação utilizando JavaScript. Aprendi os fundamentos essenciais, como variáveis, tipos de dados e estruturas de controle. Também explorei como JavaScript pode ser utilizado para criar interações dinâmicas e funcionais em aplicações web.

HTML e CSS: Ambientes de Desenvolvimento, Estrutura de Arquivos e Tags

Neste curso, adquiri habilidades essenciais para o desenvolvimento web. Aprendi a configurar ambientes de desenvolvimento adequados, entender a estrutura de arquivos de projetos web e utilizar tags HTML de forma eficiente para estruturar conteúdo.

HTML e CSS: Classes, Posicionamento e Flexbox

Durante este curso, explorei técnicas avançadas de estilização usando CSS. Aprofundei-me no uso de classes para aplicar estilos específicos a elementos HTML, dominando também técnicas de posicionamento para criar layouts complexos. Além disso, aprendi a utilizar Flexbox para criar designs responsivos e adaptáveis.
